# Library Manager
## Description
Simple implementation of a library system applying concepts like classes and objects. There are two classes:
- Book
- LibraryManager (main class)
Book structure:
- Author
- Title
- Publication Year
- Status
Book methods:
- Borrow
- Return
- Show information
------------------V2----------------------
New features:
- Implementation of lists to store multiple books
- Menu with 5 options:
  - Add book
  - Show books
  - Search book by name
  - Borrow book
  - Return book
  - Exit
## Technologies
- Java 23
- IntelliJ IDEA Community
## How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run `LibraryManager.java`.
