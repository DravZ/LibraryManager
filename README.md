# Library Manager
## Description
Simple implementation of a library system applying concepts like classes and objects. There are two classes:
- Book
- LibraryManager (main class)
Book structure:
- Author
- Title
- Publication Year
- Status
Book methods:
- Borrow
- Return
- Show information
## Technologies
- Java 23
- IntelliJ IDEA Community
## How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run `LibrayManager.java`.
